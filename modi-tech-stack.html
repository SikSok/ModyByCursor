技术栈与依赖

摩的 App · 技术栈版本与依赖

mody-app（C 端一体化）+ 服务端 + 管理端 Web · 原 user-app/driver-app 暂保留

说明

以下版本与依赖与当前仓库 package.json 一致，用于统一开发/构建环境与后续升级参考。C 端已整合为 mody-app（乘客/司机身份切换）；user-app、driver-app 目录暂保留、后续再删。

  • server - Express + TypeScript，MySQL + Sequelize，JWT / bcrypt
  • admin-web - React 18 + Vite 5 + TypeScript，Ant Design 5 + Pro Components
  • mody-app - React Native 0.74 + React 18，TypeScript（C 端主应用）
  • user-app / driver-app - 同上 RN 技术栈（暂保留，仅作参考）
