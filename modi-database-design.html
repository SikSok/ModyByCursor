数据库设计

摩迪 App 数据库表结构设计

MySQL + Sequelize ORM · 基于需求文档设计

设计说明

本数据库设计支持摩迪 App 核心功能（C 端为 mody-app 一体化，服务端 + 管理端）：用户注册登录、司机注册审核、实时定位上报、附近司机查询、短信验证码等。采用 MySQL 数据库，通过 Sequelize ORM 进行数据操作；服务端启动时 sequelize.sync({ alter: true }) 自动建表/补字段（开发与生产均执行），模型变更后重启即可生效。

  • admins - 管理员表，存储管理端账号、角色权限，用于司机审核与运营后台
  • users - 用户（乘客）表，存储用户基本信息
  • drivers - 司机表，存储司机信息、审核状态、接客状态
  • driver_locations - 司机定位记录表，存储司机实时位置（每30秒上报）
  • verification_codes - 短信验证码表，存储验证码用于注册/登录校验

数据表结构

🛡️
admins - 管理员表
存储管理端账号、角色权限，用于司机审核、用户管理、数据看板等后台功能
字段名 类型 约束 说明
id INT UNSIGNED PRIMARY KEY NOT NULL AUTO_INCREMENT 管理员ID，主键
username VARCHAR(50) UNIQUE NOT NULL INDEX 用户名，唯一索引，用于登录
password_hash VARCHAR(255) NOT NULL 密码哈希值（bcrypt加密）
name VARCHAR(50) NULL 管理员姓名/昵称，可选
email VARCHAR(100) NULL 邮箱，可选
role ENUM('super_admin', 'admin', 'operator') NOT NULL DEFAULT 'operator' INDEX 角色：super_admin-超管，admin-管理员，operator-运营
status ENUM('active', 'inactive') NOT NULL DEFAULT 'active' INDEX 状态：active-正常，inactive-禁用
created_at DATETIME NOT NULL 创建时间
updated_at DATETIME NOT NULL 更新时间 
-- Sequelize 模型定义示例
const Admin = sequelize.define('Admin', {
  id: { type: DataTypes.INTEGER.UNSIGNED, primaryKey: true, autoIncrement: true },
  username: { type: DataTypes.STRING(50), unique: true, allowNull: false },
  password_hash: { type: DataTypes.STRING(255), allowNull: false },
  name: { type: DataTypes.STRING(50), allowNull: true },
  email: { type: DataTypes.STRING(100), allowNull: true },
  role: { 
    type: DataTypes.ENUM('super_admin', 'admin', 'operator'),
    defaultValue: 'operator',
    allowNull: false
  },
  status: { 
    type: DataTypes.ENUM('active', 'inactive'),
    defaultValue: 'active',
    allowNull: false
  },
}, {
  tableName: 'admins',
  timestamps: true,
  createdAt: 'created_at',
  updatedAt: 'updated_at'
});

users - 用户表
存储乘客（用户）的基本信息，支持注册、登录、个人中心等功能
字段名 类型 约束 说明
id INT UNSIGNED PRIMARY KEY NOT NULL AUTO_INCREMENT 用户ID，主键
phone VARCHAR(11) UNIQUE NOT NULL INDEX 手机号（11位），唯一索引，用于登录
password_hash VARCHAR(255) NOT NULL 密码哈希值（bcrypt加密）
name VARCHAR(50) NULL 用户昵称，可选
avatar VARCHAR(255) NULL 头像URL，可选
status TINYINT NOT NULL DEFAULT 1 状态：1-正常，0-禁用
created_at DATETIME NOT NULL 创建时间
updated_at DATETIME NOT NULL 更新时间 
-- Sequelize 模型定义示例
const User = sequelize.define('User', {
  id: { type: DataTypes.INTEGER.UNSIGNED, primaryKey: true, autoIncrement: true },
  phone: { type: DataTypes.STRING(11), unique: true, allowNull: false },
  password_hash: { type: DataTypes.STRING(255), allowNull: false },
  name: { type: DataTypes.STRING(50), allowNull: true },
  avatar: { type: DataTypes.STRING(255), allowNull: true },
  status: { type: DataTypes.TINYINT, defaultValue: 1, allowNull: false },
}, {
  tableName: 'users',
  timestamps: true,
  createdAt: 'created_at',
  updatedAt: 'updated_at'
});
🛵
drivers - 司机表
存储司机信息、审核状态、接客状态，支持司机注册、审核、状态切换等功能
字段名 类型 约束 说明
id INT UNSIGNED PRIMARY KEY NOT NULL AUTO_INCREMENT 司机ID，主键
phone VARCHAR(11) UNIQUE NOT NULL INDEX 手机号（11位），唯一索引，用于登录
password_hash VARCHAR(255) NOT NULL 密码哈希值（bcrypt加密）
name VARCHAR(50) NOT NULL 司机姓名
avatar VARCHAR(255) NULL 头像URL，可选
id_card VARCHAR(18) NULL 身份证号（用于审核）
license_plate VARCHAR(20) NULL 车牌号
vehicle_type VARCHAR(20) NULL 车辆类型（如：摩托车、电动车）
status ENUM('pending', 'approved', 'rejected') NOT NULL DEFAULT 'pending' INDEX 审核状态：pending-待审核，approved-已通过，rejected-已驳回
is_available BOOLEAN NOT NULL DEFAULT false INDEX 接客状态：true-空闲可接客，false-接客中/不可用
last_location_id INT UNSIGNED NULL FK 最新定位记录ID（关联 driver_locations），用于快速查询
created_at DATETIME NOT NULL 创建时间
updated_at DATETIME NOT NULL 更新时间 
-- Sequelize 模型定义示例
const Driver = sequelize.define('Driver', {
  id: { type: DataTypes.INTEGER.UNSIGNED, primaryKey: true, autoIncrement: true },
  phone: { type: DataTypes.STRING(11), unique: true, allowNull: false },
  password_hash: { type: DataTypes.STRING(255), allowNull: false },
  name: { type: DataTypes.STRING(50), allowNull: false },
  avatar: { type: DataTypes.STRING(255), allowNull: true },
  id_card: { type: DataTypes.STRING(18), allowNull: true },
  license_plate: { type: DataTypes.STRING(20), allowNull: true },
  vehicle_type: { type: DataTypes.STRING(20), allowNull: true },
  status: { 
    type: DataTypes.ENUM('pending', 'approved', 'rejected'), 
    defaultValue: 'pending', 
    allowNull: false 
  },
  is_available: { type: DataTypes.BOOLEAN, defaultValue: false, allowNull: false },
  last_location_id: { type: DataTypes.INTEGER.UNSIGNED, allowNull: true },
}, {
  tableName: 'drivers',
  timestamps: true,
  createdAt: 'created_at',
  updatedAt: 'updated_at'
});
📍
driver_locations - 司机定位记录表
存储司机实时定位数据，每30秒上报一次，用于计算附近司机（Haversine算法）
字段名 类型 约束 说明
id INT UNSIGNED PRIMARY KEY NOT NULL AUTO_INCREMENT 定位记录ID，主键
driver_id INT UNSIGNED FK NOT NULL INDEX 司机ID（外键关联 drivers.id）
latitude DECIMAL(10, 8) NOT NULL 纬度（-90 到 90）
longitude DECIMAL(11, 8) NOT NULL 经度（-180 到 180）
accuracy DECIMAL(8, 2) NULL 定位精度（米），可选
created_at DATETIME NOT NULL INDEX 定位上报时间（用于查询最新定位） 
-- Sequelize 模型定义示例
const DriverLocation = sequelize.define('DriverLocation', {
  id: { type: DataTypes.INTEGER.UNSIGNED, primaryKey: true, autoIncrement: true },
  driver_id: { 
    type: DataTypes.INTEGER.UNSIGNED, 
    allowNull: false,
    references: { model: 'drivers', key: 'id' },
    onDelete: 'CASCADE'
  },
  latitude: { type: DataTypes.DECIMAL(10, 8), allowNull: false },
  longitude: { type: DataTypes.DECIMAL(11, 8), allowNull: false },
  accuracy: { type: DataTypes.DECIMAL(8, 2), allowNull: true },
}, {
  tableName: 'driver_locations',
  timestamps: true,
  createdAt: 'created_at',
  updatedAt: false
});

// 关联关系
DriverLocation.belongsTo(Driver, { foreignKey: 'driver_id' });
Driver.hasMany(DriverLocation, { foreignKey: 'driver_id' });

verification_codes - 短信验证码表
存储短信验证码，用于用户/司机注册、登录时的验证码校验（阿里云短信服务）
字段名 类型 约束 说明
id INT UNSIGNED PRIMARY KEY NOT NULL AUTO_INCREMENT 验证码记录ID，主键
phone VARCHAR(11) NOT NULL INDEX 手机号（11位）
code VARCHAR(6) NOT NULL 验证码（6位数字）
type ENUM('register', 'login') NOT NULL 验证码类型：register-注册，login-登录
expires_at DATETIME NOT NULL INDEX 过期时间（通常5-10分钟）
used BOOLEAN NOT NULL DEFAULT false INDEX 是否已使用：true-已使用，false-未使用
created_at DATETIME NOT NULL 创建时间 
-- Sequelize 模型定义示例
const VerificationCode = sequelize.define('VerificationCode', {
  id: { type: DataTypes.INTEGER.UNSIGNED, primaryKey: true, autoIncrement: true },
  phone: { type: DataTypes.STRING(11), allowNull: false },
  code: { type: DataTypes.STRING(6), allowNull: false },
  type: { 
    type: DataTypes.ENUM('register', 'login'), 
    allowNull: false 
  },
  expires_at: { type: DataTypes.DATE, allowNull: false },
  used: { type: DataTypes.BOOLEAN, defaultValue: false, allowNull: false },
}, {
  tableName: 'verification_codes',
  timestamps: true,
  createdAt: 'created_at',
  updatedAt: false
});

关键索引与查询优化

表名 索引字段 索引类型 用途
users phone UNIQUE INDEX 快速登录查询
drivers phone UNIQUE INDEX 快速登录查询
drivers status INDEX 管理端查询待审核司机（WHERE status='pending'）
drivers is_available INDEX 查询可用司机（WHERE is_available=true）
driver_locations driver_id INDEX 查询某司机的定位历史
driver_locations created_at INDEX 查询最新定位（ORDER BY created_at DESC）
verification_codes phone, expires_at, used 复合索引 验证码校验查询（WHERE phone=? AND expires_at>NOW() AND used=false）
