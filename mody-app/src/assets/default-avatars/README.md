# 默认司机头像

本目录放置 3 张**男版**「摩的司机」风格默认头像（可爱 / 友好 / 憨厚），供乘客端司机列表、司机信息弹窗与司机端个人中心在未设置头像时使用。

**风格参考**：圆脸、柔和、干净、简单背景的 3D 卡通头像。

- `default-driver-1.png`：**可爱** — 偏年轻、圆脸、微笑，可略带腮红
- `default-driver-2.png`：**友好** — 中年感、自然笑容、眼神亲切
- `default-driver-3.png`：**憨厚** — 略年长、朴实、微微笑或沉稳

可带一点「摩的」元素（如安全帽、简洁领口），不要复杂。

**格式与体积（农村/弱网）**：
- 格式：优先 **WebP**（若 RN 不支持再使用压缩 PNG）
- 尺寸：128×128 或 200×200
- 体积：单张 &lt; 15KB，三张合计 &lt; 50KB

当前为占位图，待替换为正式默认头像时可改为 `default-driver-1.webp` 等，并同步修改 `src/utils/defaultAvatars.ts` 中的 require 路径。
